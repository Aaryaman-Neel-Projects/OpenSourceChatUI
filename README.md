# Sypha Chat

A minimal web chat app built with **React**, **TypeScript**, and **Tailwind CSS**.  
Users can choose a **prompt type** (Freeform, JSON, XML, Markdown, YAML), edit the template, and chat with a mock LLM assistant.

---

## Features

- Choose from multiple prompt types  
- Model selector (GPT, Claude, Gemini, etc.)  
- Auto-filled templates per type  
- Editable prompt area  
- Mock assistant replies  
- Saves prompt type & content in local storage  

---

## Tech Stack

- React + TypeScript  
- Tailwind CSS  
- Vite
